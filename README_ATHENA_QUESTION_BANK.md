# ATHENA QUESTION BANK

Produto final do ecossistema Athena para:

1. Montar banco de questões a partir de provas ENADE, ENARE e ENAMED.
2. Montar banco de planos de aula a partir dos planos inseridos.
3. Relacionar questões com aulas, assuntos, áreas e competências.
4. Exportar questões em Word por aula, área e quantidade.
5. Gerar estatísticas de provas.
6. Produzir simulados personalizados por área, assunto e número de questões.

## Fluxo

Provas inseridas:
data/entrada/provas/

Planos inseridos:
data/entrada/planos_aula/

Banco de questões:
data/banco_questoes/

Banco de planos:
data/banco_planos/

Saídas:
outputs/word/
outputs/relatorios/
outputs/simulados/
